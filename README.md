# Task 11 - File Handling (Java)

## Description
This is a Java console application that stores and retrieves user details
using file handling concepts.

## Features
- Accepts user name, email, and phone number
- Stores data in a text file
- Appends data without overwriting
- Reads and displays stored data
- Handles file creation and exceptions

## Technologies Used
- Java
- FileReader & FileWriter
- BufferedReader & BufferedWriter

## How to Run
javac UserDataStorage.java  
java UserDataStorage
